Collaboration policy: You are encouraged to collaborate on homework provided you follow the spirit of the 50 ft rule

AI policy: We will occasionally write code using AI together in class. However, AI is not allowed on homeworks, and not recommended when you are practicing coding on your own. There is preliminary evidence that using LLMs from the get-go is really bad for learning how to code as a beginner (1, 2)

Online resources policy: Online resources, such as StackOverflow are permitted, as long as you do not copy/paste/transcribe code directly from such sources

Individual tutoring: You can schedule free individual tutoring sessions with a peer tutor here. If you have questions about finding a tutor, please contact Madison Kelsey (mjk5)
